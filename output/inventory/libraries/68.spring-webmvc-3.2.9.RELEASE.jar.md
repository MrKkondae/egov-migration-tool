## 2. 라이브러리 개요

### 주요 기능

* DispatcherServlet 제공
* Controller 처리
* URL Mapping 처리
* View Resolver 처리
* REST Controller 처리
* Data Binding 처리
* Validation 처리

### 사용 목적

`spring-webmvc`는 Spring MVC 프레임워크의 핵심 라이브러리이다.

전자정부프레임워크 기반 웹 시스템의 Controller, URL Mapping, 화면 처리, REST 서비스 대부분이 이 라이브러리를 기반으로 동작한다.

전자정부 프로젝트에서는 다음과 같은 기능에 사용된다.

* Controller 처리
* 화면 이동
* REST API 처리
* 파일 다운로드
* Validation 처리
* Exception 처리

### 대표 사용 예시

#### Controller

```java id="d2v8km"
@Controller
@RequestMapping("/user")
public class UserController {

    @RequestMapping("/list.do")
    public String selectUserList() {

        return "user/list";
    }
}
```

#### REST Controller

```java id="f5p7tw"
@RestController
public class ApiController {

    @GetMapping("/api/users")
    public List<UserVO> selectUserList() {

        return service.selectUserList();
    }
}
```

#### DispatcherServlet 설정

```xml id="m9t3xr"
<servlet>
    <servlet-name>action</servlet-name>
    <servlet-class>
        org.springframework.web.servlet.DispatcherServlet
    </servlet-class>
</servlet>
```

## 4. eGovFrame 4.3 영향도 분석

### 전환 대상 여부

* 대상

### 전환 사유

`spring-webmvc-3.2.9.RELEASE.jar`는 Spring Framework 3.2 기반 MVC 프레임워크이다.

eGovFrame 4.3은 Spring Framework 5.3.37 기반으로 구성되므로 반드시 업그레이드해야 한다.

특히 `spring-webmvc`는 전자정부프레임워크 웹 애플리케이션의 핵심 실행 구조를 담당하므로 영향도가 가장 높은 라이브러리 중 하나이다.

다음 영역에 직접적인 영향을 준다.

* Controller
* URL Mapping
* View Resolver
* Validation
* REST 서비스
* File Upload/Download
* Exception 처리

### 영향 범위

### 비고

* eGovFrame 4.3 전환 시 가장 핵심적인 전환 대상 라이브러리 중 하나이다.
* Controller, URL Mapping, View Resolver, Validation, REST 서비스 등 웹 애플리케이션 전반에 영향을 준다.
* 사용자의 전환 프로젝트 관점에서는 `spring-webmvc`, `spring-web`, `spring-context`, `spring-beans`, `spring-core`를 하나의 **Spring MVC 전환 그룹**으로 관리하는 것이 바람직하다.
* `@Controller`, `@RequestMapping`, `@RestController`는 대부분 유지 가능하지만 실제 URL 호출 및 화면 이동 검증이 반드시 필요하다.
* 전환 후에는 전체 메뉴, CRUD, 파일 업로드/다운로드, REST API, 예외처리 기능에 대한 회귀 테스트를 수행해야 한다.
* 영향도 기준으로는 현재까지 분석한 Spring 라이브러리 중 `spring-core`, `spring-context`, `spring-webmvc`, `spring-tx`가 최상위 우선순위 전환 대상으로 분류된다.
