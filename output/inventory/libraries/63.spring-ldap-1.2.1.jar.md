## 2. 라이브러리 개요

### 주요 기능

* LDAP 서버 조회
* LDAP 사용자 인증
* LDAP 데이터 검색
* LDAP 데이터 등록 및 수정
* LDAP Template 제공

### 사용 목적

`spring-ldap`는 LDAP(Lightweight Directory Access Protocol) 서버와 연계하기 위한 Spring 기반 라이브러리이다.

전자정부 프로젝트에서는 다음과 같은 기능에 활용될 수 있다.

* 사내 LDAP 인증
* Active Directory 연계
* 사용자 정보 조회
* 조직도 정보 조회
* SSO 연계

### 대표 사용 예시

```java id="a7y1qr"
LdapTemplate ldapTemplate =
    new LdapTemplate(contextSource);

List users =
    ldapTemplate.search(
        "",
        "(uid=test)",
        new UserMapper()
    );
```

## 4. eGovFrame 4.3 영향도 분석

### 전환 대상 여부

* 검토 대상

### 전환 사유

`spring-ldap-1.2.1.jar`는 Spring Framework 핵심 라이브러리가 아니라 LDAP 연계용 확장 라이브러리이다.

LDAP 또는 Active Directory를 사용하지 않는 프로젝트라면 실제 사용되지 않을 가능성이 있다.

다만 LDAP 인증 또는 조직도 연계 기능이 존재하는 경우 Spring Framework 5.3 환경과의 호환성 검토가 필요하며, 최신 Spring LDAP 버전 적용 여부를 검토해야 한다.

### 비고

* eGovFrame 4.3 전환 필수 라이브러리는 아니다.
* 실제 프로젝트에서 LDAP 인증, 조직도 연계, Active Directory 연계 기능이 없는 경우 제거 후보가 될 수 있다.
* LDAP 기능을 사용하는 경우에는 Spring Framework 업그레이드 이후 LDAP 인증 및 조회 기능에 대한 통합 테스트가 필요하다.
* 전자정부프레임워크 전환 관점에서는 Spring Core 계열(`spring-core`, `spring-beans`, `spring-context`, `spring-aop`, `spring-expression`, `spring-jdbc`)보다 우선순위가 낮다.
* 실제 프로젝트 분석 시에는 `LdapTemplate`, `ContextSource`, LDAP URL 설정 존재 여부를 반드시 확인해야 한다.
