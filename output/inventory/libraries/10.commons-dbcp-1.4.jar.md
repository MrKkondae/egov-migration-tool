## 2. 라이브러리 개요

### 주요 기능

* JDBC Connection Pool 제공
* DataSource 구현체 제공
* Connection 재사용 및 자원 관리
* Pool 크기 및 상태 관리
* Validation Query 지원

### 사용 목적

데이터베이스 연결을 효율적으로 관리하기 위한 Connection Pool 라이브러리이다.

전자정부프레임워크 3.x 환경에서는 Tomcat JNDI DataSource 또는 Spring DataSource Bean 구성 시 주로 사용되었다.

### 대표 사용 예시

```java
BasicDataSource ds = new BasicDataSource();
ds.setDriverClassName("oracle.jdbc.driver.OracleDriver");
ds.setUrl(url);
ds.setUsername(user);
ds.setPassword(password);
```

## 4. eGovFrame 4.3 영향도 분석

### 전환 대상 여부

* 대상

### 전환 사유

Commons DBCP 1.4는 유지보수가 종료(EOL)된 라이브러리이다.

eGovFrame 4.3 표준 구성에서는 Commons DBCP 1.x를 사용하지 않는다.

따라서 사용 여부를 확인한 후 Tomcat JDBC Pool 또는 Commons DBCP2로 전환하는 것이 권장된다.

### 영향 범위

| 구분      | 영향                            |
| ------- | ----------------------------- |
| 소스      | BasicDataSource 사용 시 영향       |
| 설정파일    | context.xml, server.xml 수정 가능 |
| pom.xml | 의존성 변경 필요                     |
| 운영환경    | Pool 설정 검증 필요                 |

## 6. 버전 업그레이드 검토

### AS-IS 버전

```text
commons-dbcp-1.4
```

### TO-BE 버전

권장안

```text
org.apache.tomcat:tomcat-jdbc
(Tomcat JDBC Pool)
```

대안

```text
org.apache.commons:commons-dbcp2
```

참고

```text
HikariCP
```

는 Spring Boot 환경에서 주로 사용되며 일반적인 eGovFrame 4.3 Web Project 전환 대상은 아니다.

### 주요 변경사항

* DBCP 1.x 지원 종료
* Pool 구현체 변경
* 일부 Pool 설정명 변경
* 운영환경 성능 특성 변경 가능

예)

```text
maxActive
→
maxTotal
```

```text
maxWait
→
maxWaitMillis
```

### 소스 수정 여부

* 일부 수정 필요

BasicDataSource 직접 사용 시 import 변경 필요

#### 확인 방법

* Java 소스에서 직접 사용 여부 확인
```text
grep -R "BasicDataSource" src
```

* Commons DBCP import 확인
```text
grep -R "org.apache.commons.dbcp" src
```

* Spring XML 설정 확인
```text
grep -R "BasicDataSource" resources
```

* Tomcat DataSource 설정 확인
```text
grep -R "BasicDataSourceFactory" .
```

* context.xml / server.xml 확인
```text
find . \( -name "context.xml" -o -name "server.xml" \) -exec grep -H "dbcp\|DataSourceFactory" {} \;
```


#### 수정 대상 예시

AS-IS

```java
import org.apache.commons.dbcp.BasicDataSource;
```

TO-BE
```java
import org.apache.commons.dbcp2.BasicDataSource;
```

또는

```xml
factory="org.apache.commons.dbcp.BasicDataSourceFactory"
```

↓

```xml
factory="org.apache.tomcat.jdbc.pool.DataSourceFactory"
```

### 비고

전자정부프레임워크 4.3 전환 시 HikariCP로 변경하는 것으로 단정할 수 없다.

프로젝트 내 BasicDataSource 직접 사용 여부를 우선 확인한 후 최종 전환 방안을 결정하는 것이 바람직하다.
