## 2. 라이브러리 개요

### 주요 기능

* OZ Report 데이터 암호화
* UTF-8 기반 데이터 인코딩 처리
* 리포트 서버와 연계되는 암호화 데이터 생성

### 사용 목적

OZ Report 솔루션 사용 시 리포트 조회, 출력, 다운로드 과정에서 전달되는 데이터의 암호화 및 인코딩 처리를 수행한다.

전자정부프레임워크 자체 기능과는 무관하며, OZ Report 제품에 종속적인 라이브러리이다.

### 대표 사용 예시

```java
import oz.framework.enc.OZEncryption;

String encValue = OZEncryption.encrypt(value);
```

또는

```java
import ozenc_utf8.OzEnc;
```

※ 실제 패키지명은 OZ Report 버전에 따라 상이할 수 있음

## 4. eGovFrame 4.3 영향도 분석

### 전환 대상 여부

* 대상아님

### 전환 사유

ozenc_utf8.jar는 전자정부프레임워크 라이브러리가 아니며, Spring Framework 및 eGovFrame Runtime과 직접적인 연관성이 없다.

따라서 eGovFrame 3.x → 4.3 전환 과정에서 패키지 변경, Maven 변경, Spring 설정 변경 등의 대상이 아니다.

다만 OZ Report 버전 변경 또는 JDK 변경 시 별도 호환성 검토가 필요하다.

### 영향 범위

| 구분      | 영향 |
| ------- | -- |
| 소스      | 낮음 |
| 설정파일    | 없음 |
| pom.xml | 없음 |
| 운영환경    | 중간 |

---
