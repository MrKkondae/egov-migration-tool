> `exportfile-2.0.jar`는 전자정부프레임워크 표준 라이브러리가 아니며, 공개적으로 널리 알려진 오픈소스 라이브러리도 아닙니다.

>
> 현재 JAR 내부 분석 결과가 없는 상태이므로 **확인 필요 라이브러리**로 분류합니다.

## 2. 라이브러리 개요

### 주요 기능

예상 기능

* Excel 파일 생성
* CSV 파일 Export
* TXT 파일 Export
* 대용량 데이터 다운로드
* 화면 조회 결과 다운로드

### 사용 목적

업무 시스템에서 조회 결과를 파일로 저장하기 위한 기능을 제공하는 라이브러리로 추정된다.

주로 다음과 같은 화면에서 사용된다.

* 엑셀 다운로드
* CSV 다운로드
* 통계자료 추출
* 대량 데이터 Export

### 대표 사용 예시

```java
ExportFile export = new ExportFile();

export.addRow(data);

export.write(response);
```

실제 API는 JAR 내부 확인이 필요하다.

## 4. eGovFrame 4.3 영향도 분석

### 전환 대상 여부

**확인 필요**

### 전환 사유

전자정부프레임워크 4.3은 Export 기능과 직접적인 연관이 없다.

따라서 다음 중 하나에 해당한다.

#### Case 1

프로젝트 자체 라이브러리

→ 그대로 사용 가능

#### Case 2

구형 POI 기반 라이브러리

→ Apache POI 업그레이드 필요

#### Case 3

JDK 내부 API 사용

→ JDK 17 환경에서 수정 필요
