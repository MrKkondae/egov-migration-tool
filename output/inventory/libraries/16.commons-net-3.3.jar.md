## 2. 라이브러리 개요

### 주요 기능

* FTP / FTPS 클라이언트 제공
* Telnet 클라이언트 제공
* SMTP / POP3 / IMAP 프로토콜 지원
* TFTP 지원
* NTP(Network Time Protocol) 지원
* Whois 조회 기능
* Finger 프로토콜 지원

### 사용 목적

Apache Commons Net은 다양한 네트워크 프로토콜을 Java 애플리케이션에서 쉽게 사용할 수 있도록 제공하는 라이브러리이다.

전자정부프레임워크 기반 시스템에서는 주로 다음과 같은 용도로 사용된다.

* FTP 서버 파일 업로드/다운로드
* 외부 기관 파일 연계
* 배치 프로그램 파일 송수신
* 메일 서버 연계
* 시간 동기화(NTP)

### 대표 사용 예시

```java
FTPClient ftp = new FTPClient();

ftp.connect("ftp.example.com");
ftp.login("user", "password");

InputStream input = new FileInputStream("sample.txt");

ftp.storeFile("/upload/sample.txt", input);

ftp.logout();
ftp.disconnect();
```

## 4. eGovFrame 4.3 영향도 분석

### 전환 대상 여부

* 제외

### 전환 사유

Commons Net은 전자정부프레임워크와 독립적인 Apache 네트워크 라이브러리이다.

전자정부프레임워크 4.3에서는 Commons Net을 대체하는 표준 라이브러리를 제공하지 않으므로 직접 사용 가능하다.

따라서:

* eGovFrame 3.x → 4.3 전환 대상 아님
* DAO, Service, Controller 구조 변경과 무관
* Spring Framework 5.3 업그레이드와도 직접 영향 없음

### 영향 범위

| 구분      | 영향               |
| ------- | ---------------- |
| 소스      | 없음               |
| 설정파일    | 없음               |
| pom.xml | 버전 업그레이드 검토      |
| 운영환경    | FTP/TLS 설정 확인 필요 |

## 6. 버전 업그레이드 검토

### AS-IS 버전

```text
commons-net 3.3
```

### TO-BE 버전

```text
commons-net 3.11.x 이상 권장
```

### 주요 변경사항

3.3 → 3.11 계열 주요 변화

* TLS/SSL 보안 기능 강화
* IPv6 지원 개선
* FTP 연결 안정성 향상
* 버그 수정 및 성능 개선
* 최신 JDK 호환성 개선

### 소스 수정 여부

* 대부분 없음
* FTPSClient 사용 시 일부 TLS 설정 확인 필요

예시:

```java
FTPSClient client = new FTPSClient("TLS");
```

일부 구버전 서버와의 호환성 검토가 필요할 수 있다.

## 7. 위험요소

### 보안

* Commons Net 3.3은 매우 오래된 버전
* FTP 평문 통신 사용 시 보안 위험 존재
* FTPS 또는 SFTP 사용 여부 검토 권장
* 최신 버전 적용 시 알려진 취약점 및 버그 수정 효과 기대

### 운영

* 외부 FTP 서버 정책 변경 시 영향 가능
* 네트워크 타임아웃 설정 확인 필요

### 호환성

| 항목            | 영향    |
| ------------- | ----- |
| JDK 17        | 문제 없음 |
| Spring 5.3    | 문제 없음 |
| eGovFrame 4.3 | 문제 없음 |

### 비고

* Commons Net은 전자정부프레임워크 의존 라이브러리가 아닌 일반 Apache 네트워크 라이브러리이다.
* 전환 작업 대상은 아니지만 보안 및 JDK 17 호환성 측면에서 3.11.x 이상 업그레이드를 권장한다.
* 실제 프로젝트에서 FTPClient, FTPSClient, TelnetClient 사용 여부를 확인하여 운영 영향도를 검토하는 것이 좋다.
* **Migration Matrix 기준 분류:** "전환 제외(유지 가능) / 버전 업그레이드 권장" 항목으로 관리하는 것이 적절하다.
