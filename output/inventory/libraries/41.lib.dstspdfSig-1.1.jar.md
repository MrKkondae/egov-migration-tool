## 2. 라이브러리 개요

### 주요 기능

* PDF 전자서명 검증 또는 전자문서 위변조 검증
* PDF 서명 토큰 파싱 및 서명 검증
* 인증서, TSA 타임스탬프, PDF 생성/편집 라이브러리와의 연계 가능성

### 사용 목적

`lib.dstspdfSig-1.1.jar`는 파일명과 공개 검색 사례를 기준으로 PDF 전자서명 또는 PDF 위변조 검증 기능을 제공하는 보안 모듈로 추정된다.

일반적인 오픈소스 Maven 라이브러리와 달리 공식 Maven 좌표, 공개 API 문서, 제조사 배포 정보가 명확히 확인되지 않으므로, 전환 작업 전에 공급사, 제품명, 라이선스, 유지보수 가능 여부를 반드시 확인해야 한다.

전자정부프레임워크 4.3 전환 자체보다 PDF 전자서명 기능, 인증서 검증, TSA 타임스탬프, PDF 생성 라이브러리 변경 영향과 직접 연관될 가능성이 높다.

### 대표 사용 예시

```java
// 실제 API는 공급사 문서 확인 필요
// 공개 사례 기준으로 DSTSPDFSig 클래스 사용 가능성이 있음

DSTSPDFSig dstspdfSig = new DSTSPDFSig();
dstspdfSig.init(pdfBytes);
dstspdfSig.tokenParse();
boolean valid = dstspdfSig.tokenVerify();
```

위 코드는 공개 사례를 기준으로 한 추정 예시이며, 실제 프로젝트 적용 여부는 소스 검색 및 공급사 문서로 확인해야 한다.

## 4. eGovFrame 4.3 영향도 분석

### 전환 대상 여부

* 검토

### 전환 사유

본 라이브러리는 전자정부프레임워크 표준 실행환경 라이브러리라기보다 PDF 전자서명 또는 인증서 검증을 위한 별도 공급사 모듈로 추정된다.

따라서 단순 버전업 또는 일괄 제거 대상이 아니며, 다음 항목을 기준으로 전환 여부를 확정해야 한다.

* 실제 업무 소스에서 사용 중인지 여부
* PDF 전자서명 검증 기능이 현재 업무상 필수인지 여부
* 공급사가 eGovFrame 4.3, Spring 5.3, JDK 8 이상, 대상 WAS 환경을 지원하는지 여부
* iText/OpenPDF 등 PDF 생성·편집 라이브러리 변경과의 호환 여부
* TSA 타임스탬프 클라이언트 및 인증서 검증 모듈과의 연계 여부

### 영향 범위

| 구분      | 영향 |
| ------- | ---- |
| 소스      | 사용 시 PDF 전자서명 검증 로직 영향 가능 |
| 설정파일    | 인증서 경로, TSA URL, 검증 정책, 보안 Provider 설정 영향 가능 |
| pom.xml | Maven 공개 의존성이 아닐 가능성이 높아 로컬/사설 저장소 등록 필요 |
| 운영환경    | 인증서 파일, Root CA, CRL/OCSP, TSA, 방화벽, 폐쇄망 반입 절차 영향 가능 |

## 7. 위험요소

### 보안

* 인증서 검증, 전자서명 검증, 위변조 검증과 직접 관련될 수 있어 보안 영향도가 높음
* 검증 정책 오류 시 위변조 문서 수용 또는 정상 문서 거부 가능성 있음
* Root CA, 중간 인증서, CRL/OCSP, TSA 서버 인증서 만료 영향 가능
* 공개 CVE 여부는 제품명/공급사 확인 전까지 판단 불가

### 운영

* 운영망에서 TSA, OCSP, CRL 조회가 필요한 경우 방화벽·망연계 정책 영향 가능
* 폐쇄망 반입 시 JAR, Native 모듈, 인증서 파일, 라이선스 파일 누락 가능성 있음
* PDF 생성 라이브러리 변경 시 기존 서명 검증 결과가 달라질 수 있음
* 운영 인증서와 검증 인증서가 다를 경우 환경별 테스트 필요

### 호환성

| 항목            | 영향 |
| --------------- | ---- |
| JDK 8 이상       | 공급사 호환 버전 확인 필요 |
| Spring 5.3       | 직접 영향은 낮을 수 있으나 Bean 등록/설정 방식 확인 필요 |
| eGovFrame 4.3    | 표준 라이브러리보다 업무 보안 모듈 호환성 검증 필요 |
| WAS              | JEUS/WebLogic/Tomcat 등 대상 WAS별 classloader 및 보안 Provider 확인 필요 |
| PDF 라이브러리    | iText/OpenPDF 등 변경 시 서명 검증 영향 가능 |
| TSA/인증서 모듈   | 타임스탬프, 인증서 체인, CRL/OCSP 연계 확인 필요 |
