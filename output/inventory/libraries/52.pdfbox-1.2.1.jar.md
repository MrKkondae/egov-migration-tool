## 2. 라이브러리 개요

### 주요 기능

* PDF 문서 생성
* PDF 문서 읽기 및 텍스트 추출
* PDF 병합 및 분리
* PDF 암호화 및 복호화
* PDF 메타데이터 처리

### 사용 목적

애플리케이션에서 PDF 생성, 조회, 다운로드, 문서 변환 기능을 제공하기 위해 사용된다.

전자정부 프로젝트에서는 다음과 같은 기능에 활용된다.

* 증명서 출력
* 인사기록 출력
* 급여명세서 생성
* 보고서 PDF 생성
* 문서 다운로드 서비스

### 대표 사용 예시

```java
PDDocument document = PDDocument.load(file);

PDFTextStripper stripper = new PDFTextStripper();

String text = stripper.getText(document);

document.close();
```

## 4. eGovFrame 4.3 영향도 분석

### 전환 대상 여부

* 대상

### 전환 사유

pdfbox-1.2.1은 오래된 버전으로 현재 유지보수가 종료되었으며 최신 보안 패치가 적용되지 않은 상태이다.

eGovFrame 4.3 자체와 직접적인 연관은 없으나 JDK 8 이상 환경 및 최신 오픈소스 적용 기준에 맞추기 위해 버전 업그레이드가 권장된다.

### 영향 범위

| 구분      | 영향 |
| ------- | -- |
| 소스      | 높음 |
| 설정파일    | 낮음 |
| pom.xml | 높음 |
| 운영환경    | 낮음 |

---

## 5. 전환 시 검토사항

### 변경 필요 사항

* PDFBox 1.x → 2.x 업그레이드
* Maven Dependency 변경
* PDF API 변경사항 반영
* PDF 생성 기능 검증
* PDF 다운로드 기능 검증
* 한글 폰트 처리 검증
* PDF 병합 및 텍스트 추출 기능 검증

### 자동 변환 가능 여부

| 구분          | 가능 여부 |
| ----------- | ----- |
| OpenRewrite | 일부 가능 |
| Python Rule | 가능    |
| LLM 보정      | 필요    |

### 예상 난이도

* 중간
