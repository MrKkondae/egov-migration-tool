## 2. 라이브러리 개요

### 주요 기능

* 파일 복사, 이동, 삭제
* 디렉토리 생성 및 탐색
* Stream 처리 유틸리티 제공
* 파일 인코딩 변환
* 파일 비교 및 체크섬 계산
* FileFilter 제공

### 사용 목적

Java 기본 IO API(File, InputStream, OutputStream)의 반복 코드를 줄이고 파일 처리 기능을 단순화하기 위해 사용된다.

전자정부프레임워크 3.x 기반 프로젝트에서는 다음과 같은 용도로 자주 사용된다.

* 첨부파일 업로드/다운로드
* 배치 파일 처리
* 로그 파일 관리
* 임시 파일 생성 및 삭제
* 파일 내용 읽기/쓰기

### 대표 사용 예시

```java
File src = new File("a.txt");
File dest = new File("b.txt");

FileUtils.copyFile(src, dest);

String text = FileUtils.readFileToString(
    src,
    "UTF-8"
);

FileUtils.deleteQuietly(src);
```

## 4. eGovFrame 4.3 영향도 분석

### 전환 대상 여부

* 제외

### 전환 사유

Commons IO는 전자정부프레임워크 종속 라이브러리가 아니라 일반 Apache 유틸리티 라이브러리이다.

eGovFrame 3.x → 4.3 전환 과정에서 프레임워크 구조 변경과 직접적인 관련이 없으므로 필수 전환 대상은 아니다.

다만 Commons IO 2.2는 매우 오래된 버전으로 알려진 보안 취약점 및 최신 JDK 지원 측면에서 업그레이드 검토가 필요하다.

### 영향 범위

| 구분      | 영향 |
| ------- | -- |
| 소스      | 일부 |
| 설정파일    | 없음 |
| pom.xml | 있음 |
| 운영환경    | 낮음 |

## 6. 버전 업그레이드 검토

### AS-IS 버전

```text
Commons IO 2.2
```

### TO-BE 버전

```text
Commons IO 2.15.x 이상
(전자정부프레임워크 4.3 / Spring 5 / JDK17 환경 권장)
```

### 주요 변경사항

#### Charset 파라미터 의무화

2.2에서는 다음과 같은 코드가 가능하다.

```java
String text =
    FileUtils.readFileToString(file);
```

최신 버전에서는 Charset 지정이 필요하다.

```java
String text =
    FileUtils.readFileToString(
        file,
        StandardCharsets.UTF_8
    );
```

---

#### IOUtils API 변경

기존

```java
IOUtils.toString(inputStream);
```

권장

```java
IOUtils.toString(
    inputStream,
    StandardCharsets.UTF_8
);
```

---

#### Path 지원 확대

최신 버전에서는 Java NIO Path 기반 API 지원이 강화되었다.

```java
Path path = Paths.get("sample.txt");
```

## 7. 위험요소

### 보안

Commons IO 2.2는 2012년 출시 버전으로 매우 오래되었으며 여러 보안 취약점 수정이 반영되지 않은 상태이다.

최신 버전 사용 권장.

### 운영

* 대용량 파일 처리 시 메모리 사용량 검토 필요
* 업로드 모듈과 연계 시 성능 테스트 필요

### 호환성

| 항목            | 영향       |
| ------------- | -------- |
| JDK 17        | 업그레이드 권장 |
| Spring 5.3    | 문제 없음    |
| eGovFrame 4.3 | 문제 없음    |

### 비고

* 전자정부프레임워크 4.3 자체 전환과 직접적인 연관은 없다.
* 라이브러리 노후화 관점에서 업그레이드를 권장한다.
* AS-IS 소스에서 `FileUtils`, `IOUtils`, `FilenameUtils` 사용 현황을 추가 조사하여 실제 영향도를 확인할 필요가 있다.
