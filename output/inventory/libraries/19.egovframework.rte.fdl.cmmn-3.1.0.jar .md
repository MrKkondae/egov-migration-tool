# 2. 라이브러리 개요

## 주요 기능

* eGovFrame 공통 유틸리티 제공
* ID 생성기(egov-idgn)
* Property Service
* Trace Handler
* Exception 처리
* Reflection 기반 Bean 처리
* Resource 및 Message 처리 지원

## 사용 목적

전자정부프레임워크 3.x에서 공통 기능을 제공하는 Foundation Layer(FDL)의 핵심 라이브러리이다.

주로 다음 기능에서 사용된다.

* EgovPropertyService
* EgovIdGnrService
* TraceHandler
* EgovResourceCloseHelper
* ReflectionToStringBuilder
* 공통 예외 처리

## 대표 사용 예시

```java
@Resource(name="propertiesService")
protected EgovPropertyService propertiesService;

String pageUnit =
    propertiesService.getString("pageUnit");
```

```java
@Resource(name="egovIdGnrService")
private EgovIdGnrService egovIdGnrService;

Long id = egovIdGnrService.getNextLongId();
```

# 4. eGovFrame 4.3 영향도 분석

## 전환 대상 여부

* 대상

## 전환 사유

eGovFrame 4.3에서는 FDL 라이브러리 전체가 4.3 버전으로 업그레이드된다.

현재 사용 중인

```text
egovframework.rte.fdl.cmmn-3.1.0.jar
```

는 제거되고

```text
org.egovframe.rte:org.egovframe.rte.fdl.cmmn:4.3.0
```

으로 변경된다.

또한 Maven 좌표 체계가 변경된다.

### 변경 전

```xml
<groupId>egovframework.rte</groupId>
<artifactId>egovframework.rte.fdl.cmmn</artifactId>
<version>3.1.0</version>
```

### 변경 후

```xml
<groupId>org.egovframe.rte</groupId>
<artifactId>org.egovframe.rte.fdl.cmmn</artifactId>
<version>4.3.0</version>
```

## 영향 범위

| 구분      | 영향 |
| ------- | -- |
| 소스      | 일부 |
| 설정파일    | 일부 |
| pom.xml | 높음 |
| 운영환경    | 중간 |

# 5. 전환 시 검토사항

## 변경 필요 사항

### 1. Maven Dependency 변경

3.1 → 4.3 좌표 변경 필요

### 2. Spring Framework 버전 영향

기존

```text
Spring 3.x
```

에서

```text
Spring 5.3.x
```

로 변경됨

FDL 내부 구현도 이에 맞춰 변경됨

### 3. PropertyService 사용 확인

아래 패턴 사용 시 정상 동작 여부 검증 필요

```java
propertiesService.getString(...)
propertiesService.getInt(...)
```

### 4. ID Generator 사용 여부 확인

```java
EgovIdGnrService
```

사용 시 Bean 설정 확인 필요

## 자동 변환 가능 여부

| 구분          | 가능 여부 |
| ----------- | ----- |
| OpenRewrite | 가능    |
| Python Rule | 가능    |
| LLM 보정      | 일부 필요 |

## 예상 난이도

* 낮음 ~ 중간

# 6. 버전 업그레이드 검토

## AS-IS 버전

```text
eGovFrame FDL Common 3.1.0
```

## TO-BE 버전

```text
eGovFrame FDL Common 4.3.0
```

## 주요 변경사항

### Maven 좌표 변경

```text
egovframework.rte.*
↓
org.egovframe.rte.*
```

### Spring Framework 업그레이드

```text
Spring 3.x
↓
Spring 5.3.x
```

### 내부 공통 유틸리티 개선

* 예외 처리 개선
* Bean 관리 개선
* JDK 호환성 향상
* Spring 기반 구조 개선

## 소스 수정 여부

* 일부 수정 필요

일반적인 경우 import 자동 변경만으로 해결된다.

```java
import egovframework.rte.fdl.property.EgovPropertyService;
```

↓

```java
import org.egovframe.rte.fdl.property.EgovPropertyService;
```

## 비고

* 본 라이브러리는 전자정부프레임워크 핵심 FDL(Common) 라이브러리로 전환 제외 대상이 아니다.
* 대부분의 프로젝트에서 직접 또는 간접적으로 사용된다.
* 3.1 → 4.3 전환 시 Maven 좌표 변경과 import 패키지 변경이 필수이다.
* OpenRewrite 및 Python Rule로 자동 변환이 가능한 대표적인 라이브러리이다.
* 실제 프로젝트에서는 `EgovPropertyService`, `EgovIdGnrService` 사용 여부를 우선 확인하는 것을 권장한다.
